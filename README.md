# CRUD Funcionarios - Java

## Descripción
Aplicación desktop desarrollada en Java Swing que permite gestionar funcionarios mediante operaciones CRUD utilizando patrón DAO y manejo de excepciones.

## Tecnologías
- Java
- Maven
- MySQL
- Swing

## Funcionalidades
- Crear funcionarios
- Listar funcionarios
- Actualizar funcionarios
- Eliminar funcionarios
- Validaciones de formulario
- Uso de JComboBox para relaciones
- Visualización de nombres en lugar de IDs

## Base de datos
Scripts incluidos en carpeta /database
